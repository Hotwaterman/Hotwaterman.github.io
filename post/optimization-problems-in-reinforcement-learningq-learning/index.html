Optimization problems in reinforcement learning（Q-learning）

2020-07-18 20 min read

key concepts in RL

The main characters of RL are the agent and the environment. At every step of interaction, the agent sees a (possibly partial) observation of the state of the world, and then decides on an action to take. The agent also perceives a reward signal from the environment, a number that tells it how good or bad the current world state is. The goal of the agent is to maximize its cumulative reward, called return. Reinforcement learning methods are ways that the agent can learn behaviors to achieve its goal. To talk more specifically what RL does, we need to talk about

  1. states sss and observations ooo.
  2. action aaa and action space which is the set of all valid actions in a given environment.
  3. policy is a rule used by an agent to decide what actions to take. at∼π(⋅∣st)a_t\sim \pi(\cdot|s_t)at​∼π(⋅∣st​) .
  4. trajectory τ\tauτ is a sequence of states and actions: τ=(s0,a0,s1,a1,… )\tau =(s_0,a_0,s_1,a_1,\dots)τ=(s0​,a0​,s1​,a1​,…) .
  5. The reward function RRR is critically important in reinforcement learning. rt=R(st,at,st+1)r_t = R(s_t,a_t,s_{t+1})rt​=R(st​,at​,st+1​), sometimes it can be simplified as rt=R(st,at)r_t = R(s_t, a_t)rt​=R(st​,at​) . And normally we use the infinite-horizon discounted return R(τ)=∑t=0γtrtR(\tau) = \sum_{t=0}\gamma^t r_tR(τ)=∑t=0​γtrt​, where γ∈(0,1)\gamma \in (0,1)γ∈(0,1) .

Optimization problem and Algorithm

Here we give a very popular algorithm called Q learning, we define the optimal action-value function Q∗(s,a)Q^*(s,a)Q∗(s,a) as the maximum expected return achievable by following any policy, after seeing some sequence sss and then taking some action aaa, Q∗(s,a)=maxπE[Rt∣st=s,at=a,π]Q^*(s,a) = max_{\pi}E[R_t|s_t =s,a_t=a,\pi]Q∗(s,a)=maxπ​E[Rt​∣st​=s,at​=a,π], where π\piπ is a policy that the agent choose actions [1].

The optimal action-value function obeys an important identity known as the Bellman equation. This is based on the following intuition: if the optimal value Q∗(s,,a,)Q^*(s^,,a^,)Q∗(s,,a,) of the sequence s,s^,s, at the next time-step was known for all possible actions a,a^,a,.

Then the optimal problem is to select the action a,a^,a, maximizing the expected value of r+γQ∗(s,,a,)r+\gamma Q^*(s^,,a^,)r+γQ∗(s,,a,),

(3)Q∗(s,a)=Es,∼ε[r+γmaxa,Q∗(s,,a,)∣s,a]Q^*(s,a)=E_{s^,\sim\varepsilon}[r+\gamma max_{a^,}Q^*(s^,,a^,)|s,a] \tag{3} Q∗(s,a)=Es,∼ε​[r+γmaxa,​Q∗(s,,a,)∣s,a](3)

The basic idea behind many reinforcement learning algorithms is to estimate the action-value function, by using the Bellman equation as an iterative update, Qi+1(s,a)=E[r+γmaxa,Qi(s,,a,)∣s,a]Q_{i+1}(s,a)=E[r+\gamma max_{a^,}Q_i(s^,,a^,)|s,a]Qi+1​(s,a)=E[r+γmaxa,​Qi​(s,,a,)∣s,a]. Such value iteration algorithms converge to the optimal action-value function, Qi→Q∗Q_i \rightarrow Q^*Qi​→Q∗ as i→∞i \rightarrow \inftyi→∞ [2].

In practice, it is common to use a function approximator to estimate the action-value function. We refer to a neural network function approximator with weights θ\thetaθ as a Q-network.
