Game Theory 博弈论

博弈论是对多人且彼此相互影响的情形建模。

learning over multiple agents

每个agent的学习都会对环境和其他agent造成影响，这种同时学习意味着每个学习规则都能导致一个动态系统，且简单的rules可以导致一个系统的的复杂动作。当进行动作选择时，player不仅要考虑到他从其他player的过去行为中学习到的，还要考虑到他的动作对其他player未来动作的影响。

重复游戏被视为“学习”的特性设置：时间特质和跨时间规律性（每次涉及同一名玩家和相同的游戏）。这允许我们考虑策略，未来动作取决于过去经验。

Fictitious Play

虚拟play是一个简单的序列进程来学习游戏的价值。一个model-based learning的例子：

repeat：

观测对手实际动作，更新belief

在fictitious play中，agent相信对手与先前动作的经验分布相关的混合策略。混合策略为：

Evolutionary Game theory

博弈论继续使用即使个体可能不会做出正确合理的决定，所以抉择可能是无意识的。

许多动作涉及到种群中生物间的相互影响，所以评估其适应度，应从整体种群角度出发。


Multi-agent Reinforcement Learning

MAL困难在智能体不仅与环境交互，而且每个智能体之间也存在相互影响。若是通过把其他智能体当作环境的一部分来考虑使用单智能体的Q learning，这将破坏收束保证，导致学习的不稳定。

序列决策过程：

马尔可夫决策过程：一个决策者，多种状态
重复游戏：多个决策者，一种状态
随机游戏（马尔科夫游戏）：多个决策者，多种状态

重复游戏又叫matrix game，因为奖励函数和状态可以用一个n维数组来表示。

随机游戏：(n,S,A1...n,T,R1...n)(n,S,A_{1...n},T,R_{1...n})(n,S,A1...n​,T,R1...n​) 其中n为player的数量，S是状态集合，AiA_iAi​是对于player i的可用动作集，T是转移函数，R是奖励函数。

与MDP不同，有很多玩家进行动作选择，下一状态和奖励由联合动作决定并且每个玩家有其自己的奖励函数。

如果固定除了一个agent之外其他agent的策略，那么问题将变成MDP。

Desired properties of multi-agent learners

rationality:如果其他智能体的策略收敛于一个稳态策略，那么学习算法将收敛于一个对其他智能体的best-response

convergence：学习者收敛于一个稳态策略。
